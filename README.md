## 📋 About MediConnect

MediConnect is a healthcare appointment system designed for smooth and quick patient booking. It connects directly to Google Sheets so your booking data updates in real time. The system creates unique IDs for every appointment. Its interface adjusts automatically to your screen size, making it easy to use on any device.

This app handles appointments clearly and efficiently. You do not need special technical knowledge to use it. The user interface is straightforward and fast.

## 🛠 Using MediConnect

### Booking Appointments

- Open MediConnect and click "New Appointment"
- Fill simple form fields like patient name, date, and time
- Click "Save" to store the appointment

Each appointment will receive a unique ID automatically.

### Viewing and Editing Appointments

- Use the calendar or list view inside the app to see all bookings
- Click any appointment to update or cancel it
